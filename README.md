# BSAS CodeIgniter 4 Website

A CodeIgniter 4 starter app for the BSAS website, based on the uploaded website form PDF.

## Pages included
- Home
- About Us
- Spare Parts
- Equipments
- Services
- E-Shop
- 3R
- Support with quote form
- FAQ
- Privacy Policy

## Run locally
```bash
composer install
cp env .env
php spark serve
```

Open http://localhost:8080

## Notes
This is a ready-to-customize CodeIgniter app scaffold. Replace placeholder imagery/video blocks with final brand assets and connect the quote form to email, CRM, or database as needed.
